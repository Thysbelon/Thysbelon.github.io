I spent about two weeks migrating my website from Jekyll to Eleventy. I feel like it'll be much easier to maintain now.

I'm not sure I can write a good post about my experiences, so I'll just make a big list of tips and links.

  • If you have set a default layout, make sure to code the layout so it skips rendering for pages with "permalink: false" (or else every build that hides draft posts will fail, and manipulates post URLs with string functions, will fail)
