You can receive news via Push Notifications or Atom Feed

僕がブログに投稿する時プッシュ通知をもらう

Clicking on the button below will subscribe you to notifications sent via Firebase Cloud Messaging, a service run by Google. This will save an IndexedDB to your computer's storage containing the subscription data, and start running a service worker in the background, so you can receive notifications when this website is not open. I have not added analytics to this website.

Subscribe to Atom feed

Atom is a language similar to RSS. It is used on many webpages. By using a Reader, you can receive a syndication of recent updates from every website you follow.
