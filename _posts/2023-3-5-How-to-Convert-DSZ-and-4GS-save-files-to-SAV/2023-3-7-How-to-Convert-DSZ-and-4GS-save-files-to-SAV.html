44670's two great web emulator ports: 44gba and DS Player, both use a completely unique file extension for their save files; Here's how to convert these formats to .sav:

4GS

This is just a renamed version of the emulator Visual Boy Advance's srm save format.

To convert it to sav:

  1. Rename it to change the .4gs extension to .srm, and remove all spaces from the file name.
  2. Use gbaconv to convert

Anyway, more details on VBA's save format can be found on emugen wiki, this link also contains links to gbaconv's source code and a ready-to-use gbaconv binary for 64-bit Windows.

Compiling is the process of taking source code and converting it into an executable file that your computer can run. For users of operating systems like MacOS and Android, it shouldn't be too hard to compile gbaconv with a terminal command like cc gbaconv.c -o gbaconv. The terminal is included by default on MacOS, but Android users will have to install Termux and allow Termux to access their phone's storage.

DSZ

This is Desmume's dsv save format compressed to 7z.

To convert it to sav:

  1. Install an adblocker if possible. Android users can install extensions with Kiwi Browser or Firefox
  2. Extract the uncompressed save file with 7-Zip or this online extractor. You may have to rename the .dsz file to .7z
  3. The extracted save file wont have an extension, but it is still a dsv file. Use this website to convert it to sav (many ads; if you don't have adblock, be careful. You need to click the grey "convert" button).
